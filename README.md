## Installation
### If you use Windows
Run InstallAndLaunch.bat or Start.exe
### Otherwise
1. Install dependencies
Firstly, install python and don't forget to enable adding to PATH. Then you can go to your directory and run in cmd/bash:
```bash
pip install -r requirements.txt
```
2. Run it
```bash
python main.py
```
